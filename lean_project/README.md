# Lean Lecture Notes

This directory contains the Lean 4 project for Algebra 4 lecture notes.

## Building

To build this project, you need Lean 4 installed. Then run:

```bash
lake build
```

## Project Structure

- `lakefile.lean` - Lake build configuration
- `lean-toolchain` - Specifies the Lean version
- `Main.lean` - Main entry point
- `LectureNotes.lean` - Main lecture notes module
- `LectureNotes/` - Additional lecture note modules
